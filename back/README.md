Statistics: 
- Strenght => impacts the damage of physical attacks.
- Precision => impacts the chance to hit.
- Hp => impacts the health points.
- Mp => impacts the magic points.
- Defense => impacts the damage taken from physical attacks.
- Magic => impacts the damage of magical attacks.
- Spirit => impacts the damage taken from magical attacks.
- Evasion => impacts the chance to dodge attacks.
- Speed => impacts the turn order in battle.
